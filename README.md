# 🛰️ Internet Scavenger Dashboard

A high-density, real-time research dashboard built with Streamlit, Plotly, and HTML5 Canvas. **Internet Scavenger** aggregates and visualizes micro-trend retail intelligence across major e-commerce platforms like Amazon, Etsy, TikTok Shop, and AliExpress.

## ✨ Features

- **Spotlight Item Node:** Priority focus view on top-performing items with direct store links and telemetry indicators.
- **Interactive Plotly Analytics:** Toggleable 7-day cumulative sales trends and 24-hour velocity metrics mapped by custom day selectors.
- **Recon Orbit Canvas:** Embedded HTML5/JavaScript animated planetary telemetry module.
- **Top 10 Micro-Trends Feed:** Compact, real-time scrollable matrix tracking micro-trends outside core positions.
- **Retail Corridor Evaluation Matrix:** Responsive 5-column grid system with hover micro-interactions.
- **Dark Glassmorphism Theme:** Translucent CSS card design using backdrop filters and custom neon gradients.
