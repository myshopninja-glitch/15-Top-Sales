# Internet Scavenger Hub

A high-fidelity analytics tracking portal stylized like a Diablo 2 Lord of Destruction workspace. 

## Features
- **Dynamic Grid Navigation:** Click on any slot in the 15-item array to shift data directly to the central inspection viewport.
- **Drill-Down Analytics Tome:** Click day selectors on the interactive sales report matrix to toggle instantly between weekly run totals and active 24-hour delivery spreads.
- **Animated HTML5 Scrying Display:** Custom canvas rendering loops with real-time character matrix loops mapping querying activities.
- **Storefront Anchors:** Hyperlinks link out cleanly to live shopping destination pipelines.

## How to Deploy to Streamlit Community Cloud
1. Push `app.py` and `requirements.txt` into a public GitHub repository.
2. Visit [share.streamlit.io](https://share.streamlit.io/) and log in using your GitHub account credentials.
3. Click **"New App"**, find your target tracking repository branch, and click **Deploy**.
